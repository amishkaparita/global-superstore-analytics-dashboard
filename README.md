# Global Superstore Sales Dashboard (2021–2022)

This project is a Power BI dashboard built using the Global Superstore dataset for learning and exploring business analytics, sales insights, and data visualization concepts.

## Overview
The dashboard provides interactive insights into:
- Total revenue
- Profit margins
- Shipping costs
- Product distribution
- Regional sales performance
- Customer-based analysis
- Salesperson performance
- Monthly sales trends

## Key Features
- KPI cards for revenue, margin, shipping cost, and item count
- Interactive slicers for:
  - Customer
  - Product
  - Region
  - Salesperson
- Revenue and margin trend analysis
- Product and customer segmentation
- Region-wise performance visualization
- Monthly item sales tracking

## Tools Used
- Power BI
- DAX
- Data Visualization
- Business Analytics Concepts
